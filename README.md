# MedChat Mobile - مساعدك الطبي الذكي

**Intelligent Mobile Medical Assistant Powered by AI**  
Available in Arabic and English — Fast, Secure, and User-Friendly.

> **Important Medical Disclaimer**: This app is for informational purposes only and is NOT a substitute for professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare provider.

---
## Features

- Instant medical conversations powered by advanced AI
- Beautiful, responsive UI with full Dark Mode support
- Chat history saved locally (no server required)
- Complete Arabic language support with RTL layout
- Clean, fast, and modern codebase built with React Native + Expo
- Highly organized and easily scalable architecture
